## 🚀 How to Run

### 1. Running with Port Prompt (Interactive)
Simply run pytest without arguments. The suite will ask you for the port.
```bash
./tests/venv/bin/pytest tests/test_11_duplicate_entry.py -s
```

### 2. Running with explicit --port flag
```bash
./tests/venv/bin/pytest --port 5173
```

### 3. Running with explicit --base-url flag
```bash
./tests/venv/bin/pytest --base-url http://localhost:5173
```

### 4. Running specific modules by Marker
```bash
# Run only CRUD tests
./tests/venv/bin/pytest -m crud

# Run only E2E workflows
./tests/venv/bin/pytest -m e2e
```
